Responsive Design

We usually visit our website on the desktops or laptops. with the technology grew we get to see new devices. And every device have its own unique features. Thats where responsive web design comein picture.
Resposnsive web design is a popular technique to make the web pages adjustable according to the device specification so the same type of content look beautiful in the different kind of devices.
